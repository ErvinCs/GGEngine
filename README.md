### GGEngine - 3D Game Engine
Features
* Texturing
* MipMapping
* Fog Effect
* Lighting: World Light & Point Light
* Terrain Generation 
* HeightMaps
* Shadows

Technologies
* Java 15
* OpenGL
* LWJGL2
* JInput
